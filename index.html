Últimos Projetos

E-commerce de Produtos Saudáveis

Desenvolvi uma plataforma de e-commerce moderna e responsiva usando HTML, CSS e JavaScript. O site apresenta um carrossel interativo na página inicial, um menu de navegação que se adapta a dispositivos móveis e uma galeria de produtos.

Snake Game

Este projeto é uma implementação do clássico e popular jogo da Cobra, desenvolvido inteiramente com HTML, CSS e JavaScript. O objetivo do jogo é controlar uma cobra que se move por um campo quadriculado, comendo "alimentos" que aparecem aleatoriamente para crescer e aumentar sua pontuação. O jogo termina quando a cobra colide com as bordas do campo ou com seu próprio corpo.

Pokedex

Este projeto é uma Pokedex interativa, uma enciclopédia digital de Pokémon. Ela permite que os usuários busquem informações sobre diferentes Pokémon, visualizando sua imagem e número de identificação. O design do projeto imita o estilo de uma Pokedex dos jogos e animes, o que o torna visualmente atraente e familiar aos fãs.

Sistema de Pedidos de Pizza

Este projeto é um sistema interativo de pedidos de pizza, desenvolvido com HTML, CSS e JavaScript. A aplicação permite que os usuários visualizem um cardápio de pizzas, selecionem a quantidade, escolham o tamanho e, em seguida, vejam o resumo do pedido com os cálculos de subtotal, desconto e total.
